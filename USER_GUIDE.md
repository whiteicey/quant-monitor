## 1. 怎么打开程序

### 电脑上用

1. 从儿子发给你的文件里，找到 **`A股信号监控.exe`**
2. **双击打开**，等几秒钟，浏览器会自动弹出来
3. 看到页面就可以用了！

> 如果浏览器没有自动弹出，手动打开浏览器，地址栏输入 `http://127.0.0.1:5000` 然后回车。

### 手机上用

1. 电脑先把程序打开（上面的步骤）
2. 看一下电脑终端窗口里显示的地址，类似 `http://192.168.x.x:5000`
3. 手机连上和电脑**同一个WiFi**
4. 手机浏览器输入那个地址
5. 建议：点浏览器菜单 → "添加到主屏幕"，这样以后像打开APP一样方便

## 3. 自选股 — 你的股票清单

![自选股总览](screenshots/watchlist.png)

这是打开程序后的**默认页面**。

### 看什么

表格里每只股票显示：

| 列名 | 含义 | 怎么看 |
|------|------|--------|
| **股票** | 代码和名称 | — |
| **最新价** | 当前价格 | 交易时间内每5秒自动刷新 |
| **涨跌幅** | 今日涨跌 | **红色=涨，绿色=跌**（和炒股软件一样） |
| **多头** | 看涨信号强度 | 最高6分，越高越看涨 |
| **空头** | 看跌信号强度 | 最高6分，越高越看跌 |
| **趋势** | 当前大方向 | 上涨/下跌/震荡 |
| **信号** | 操作建议 | 见下方说明 |

### 信号含义

| 信号 | 颜色 | 意思 | 该怎么做 |
|------|------|------|---------|
| **强烈买入** | 红色底 | 多个指标同时看涨 | 可以**重点关注**，考虑买入 |
| **买入** | 普通红 | 部分指标看涨 | 可以关注，不急 |
| **观望** | 灰色 | 没有明确方向 | 不操作，等信号 |
| **卖出** | 普通绿 | 部分指标看跌 | 注意风险 |
| **强烈卖出** | 绿色底 | 多个指标同时看跌 | 认真考虑卖出或减仓 |

> **重要提醒**：信号只是参考，不是"一定涨/一定跌"。要结合自己的判断。

### 操作

- **点击某一行** → 自动跳到个股详情页，查看详细分析
- **删除** → 点行末尾的删除按钮
- **添加** → 用页面上方的搜索框，下文有详细说明

## 4. 搜索和添加股票

### 搜索股票

顶部有个搜索框（默认写着股票代码）：

1. **输入股票代码**（如 `600519`）或**名称**（如"茅台"）
2. 会自动弹出下拉列表显示搜索结果
3. **点击你要的那个** → 自动跳到个股详情并开始分析

### 添加到自选股

在自选股页面：

1. 找到页面上方的 "搜索添加" 输入框
2. 输入股票代码或名称
3. 从下拉列表里点击想添加的股票
4. 自动加入自选股列表

## 6. K线图操作

图表区从上到下有5个图表：

| 图表 | 看什么 | 重点关注 |
|------|--------|---------|
| **K线图**（最大的） | 股价走势 | 红色K线=涨，绿色K线=跌 |
| **MACD** | 趋势强弱 | 柱子由绿变红=可能要涨 |
| **RSI** | 超买超卖 | >70 过热（可能要跌），<30 过冷（可能要涨） |
| **KDJ** | 短线转折 | K线上穿D线=买入信号 |
| **OBV** | 资金流向 | 线往上=资金流入，线往下=资金流出 |

### K线图上的线

| 颜色 | 线名 | 含义 |
|------|------|------|
| 蓝色线 | 快速EMA | 短期趋势（默认6天） |
| 红色线 | 慢速EMA | 中期趋势（默认7天） |
| 灰色上线 | 布林带上轨 | 股价偏高区域 |
| 灰色下线 | 布林带下轨 | 股价偏低区域 |
| 黄色中线 | 布林带中轨 | 平均价格 |
| 紫色线 | VWAP | 成交量加权平均价（机构参考） |

### K线图上的信号标记

| 标记 | 含义 |
|------|------|
| 🔺 红色上三角（K线下方） | 买入信号 |
| 🔻 绿色下三角（K线上方） | 卖出信号 |

### 画线工具

K线图上方有三个小按钮：

| 按钮 | 用途 | 操作方法 |
|------|------|---------|
| **— 水平线** | 标记某个重要价位 | 点按钮 → 在图上点一下 |
| **╱ 趋势线** | 画斜线看趋势 | 点按钮 → 在图上点两下（起点和终点） |
| **✕ 清除** | 删除所有画的线 | 直接点 |

> 画的线会自动保存，换股票再回来还在。

### 切换周期

顶部有一排周期按钮：

| 周期 | 看什么行情 |
|------|-----------|
| **5分钟/15分钟/30分钟/1小时** | 日内短线波动 |
| **日线**（默认） | 每天一根K线，最常用 |
| **周线** | 看中期趋势 |
| **月线** | 看长期趋势 |

### 调整日期范围

- **开始日期**：默认2024-01-01，可以往前调看更多历史
- **结束日期**：默认今天

修改后点 **[分析]** 按钮生效。

## 7. 侧边栏信息解读

右边的信息栏从上到下：

### ⚖ 多空评分

最重要的综合判断！

```
多头 ████████░░ 5/6     看涨力量
空头 ██░░░░░░░░ 1/6     看跌力量
```

- 多头 ≥ 4 且空头 ≤ 1 → 偏多，可能上涨
- 空头 ≥ 4 且多头 ≤ 1 → 偏空，可能下跌
- 差不多 → 震荡，观望

### ⚡ 信号状态

| 指标 | 看什么 |
|------|--------|
| 趋势 | 上涨/下跌/震荡 |
| 强买 | 是/否（出现"是"要重点关注） |
| 强卖 | 是/否（出现"是"要注意风险） |
| 买入 | 是/否 |
| 卖出 | 是/否 |

### 📊 技术指标

| 指标 | 正常范围 | 怎么看 |
|------|---------|--------|
| RSI | 30–70 | >70 偏热，<30 偏冷 |
| MACD | 看正负 | 正数=偏多，负数=偏空 |
| 价格位置 | 0%–100% | 0%=底部，100%=顶部 |

### 🎯 关键价位

这是程序根据技术指标算出来的**参考价格**：

| 价位 | 用途 |
|------|------|
| **建议买入** | 如果想买，可以在这个价附近挂单 |
| **建议卖出** | 如果想卖，可以在这个价附近挂单 |
| **止损价位** | 如果亏损到这个价，考虑止损卖出 |
| **支撑位** | 股价往下跌时，可能在这里止跌 |
| **阻力位** | 股价往上涨时，可能在这里受阻 |

> 这些价格只是数学计算的参考，不是保证！

### 🏭 板块联动

显示这只股票属于什么行业，以及今天涨跌最猛的板块TOP10。

- **所属行业** — 这只股票的行业分类
- **板块强弱TOP10** — 今天哪些行业板块最强/最弱
- 如果你的股票所在的行业板块在TOP10里面出现，说明整个板块在动，不是个股单独在涨跌

## 8. 回测 — 看策略历史表现

![回测结果](screenshots/backtest.png)

**回测 = 用历史数据模拟交易，看如果按照某个策略买卖，能赚多少钱。**

### 怎么做

1. 确保已经选了一只股票（搜索或从自选股点进来）
2. 设好日期范围（默认看2024年至今）
3. 点击顶部的 **[回测]** 按钮
4. 等几秒，结果出来了

### 看懂回测结果

结果面板里有一排数字：

| 指标 | 含义 | 好的标准 |
|------|------|---------|
| **总收益率** | 这段时间赚了多少% | 正数=赚钱，越高越好 |
| **年化收益** | 换算成一年能赚多少% | >10% 算不错 |
| **最大回撤** | 中间最多亏过多少% | 越小越好，>-20% 要注意 |
| **胜率** | 赢的次数占比 | >50% 算不错 |
| **交易次数** | 总共买卖了几次 | 太多=频繁交易费钱，太少=不确定 |
| **盈/亏** | 赚钱次数/亏钱次数 | 赚多亏少当然好 |
| **Sharpe** | 性价比（收益÷风险） | >1 算不错，>2 很好 |

### 权益曲线图

回测结果下面有一个图：

- **蓝色面积线** = 策略的资金变化（你的钱怎么变的）
- **橙色虚线** = 买入持有基准（如果你一开始全买了一直拿着）
- **灰色虚线** = 初始资金线（100万的水平线）

**怎么看**：
- 蓝线在橙线上面 → 策略比傻持着赚得多 ✅
- 蓝线在橙线下面 → 还不如不折腾 ❌
- 蓝线波动越小越好（说明风险低）

### 交易明细表

每一笔买卖的详细记录：

| 列 | 含义 |
|----|------|
| 买入日期 | 什么时候买的 |
| 卖出日期 | 什么时候卖的 |
| 买入价/卖出价 | 买卖价格 |
| 数量 | 买了多少股 |
| 盈亏 | 这笔交易赚/亏了多少钱 |
| 收益率 | 这笔交易赚/亏了百分之多少 |
| 类型 | 为什么卖的（见下表） |

卖出类型说明：

| 类型 | 含义 |
|------|------|
| signal | 策略产生了卖出信号 |
| 止损 | 亏到了设定的止损线 |
| 止盈 | 赚到了设定的止盈线 |
| 移动止损 | 从最高点回落超过设定比例 |
| (未平仓) | 到回测结束还没卖，还拿着 |

### 选择不同策略

在右侧 **💰 回测参数** 里，"信号策略"下拉框可以换：

| 策略 | 特点 | 适合场景 |
|------|------|---------|
| **MACD金叉+RSI超买** | 默认策略，信号较准 | 通用 |
| **稳健保守** | 多条件确认才买 | 不想频繁操作 |
| **趋势跟踪** | 跟着大趋势走 | 趋势明显的股票 |
| **超跌反弹** | RSI很低时抄底 | 跌多了的股票 |
| **布林带反弹** | 碰到下轨买上轨卖 | 震荡行情 |
| **纯MACD交叉** | 金叉买死叉卖 | 经典简单 |
| **纯EMA交叉** | 快线穿慢线买卖 | 经典简单 |

> 换策略后重新点 **[回测]** 才生效。

## 9. 策略对比 — 哪个策略最好

![策略对比](screenshots/compare.png)

**一次跑多个策略，看谁最强。**

### 怎么做

1. 点击顶部 **[策略对比]** 按钮
2. 弹出策略选择窗口
3. 勾选你想对比的策略（可以全选，也可以选几个）
4. 点 **[开始对比]**
5. 等一会儿，结果出来

### 看懂对比结果

- **上方图表** — 多条彩色曲线，每条是一个策略的资金变化
  - 点击图例可以显示/隐藏某条线
  - 最后在上面的策略赚得最多
- **下方表格** — 所有策略的排名，按总收益从高到低排
  - 第一名会有金色高亮

### 用途

- 找出**过去表现最好的策略**
- 看不同策略在同一只股票上的差异
- **注意**：过去表现好不代表未来一定好！

## 10. Walk-Forward验证 — 策略是否靠谱

这是一个高级功能，用来检验策略是不是**真的有效**，还是只是碰巧在历史数据上表现好。

### 原理（简单说）

把历史数据分成5段，每段分前后两部分：
- 前面一部分 = "训练集"（相当于你先看了这段行情）
- 后面一部分 = "测试集"（相当于你没看过的新行情）

如果策略在"没看过的"数据上也能赚钱，说明策略**比较靠谱**。

### 怎么做

1. 选好股票
2. 点击顶部 **[Walk-Forward]** 按钮
3. 等一会儿

### 看懂结果

| 指标 | 含义 | 好的标准 |
|------|------|---------|
| **平均测试收益** | 在未见数据上平均赚多少 | 正数=策略有效 |
| **平均测试Sharpe** | 未见数据上的性价比 | >0.5 还行 |
| **盈利窗口** | 5段里有几段赚钱 | 3/5以上=比较稳定 |
| **一致性评分** | 稳定程度 | >60%=比较靠谱 |

下面的表格显示每一段的详细结果。

### 怎么判断

- ✅ 一致性 ≥ 60%，平均测试收益为正 → 策略还不错
- ⚠ 一致性 40-60% → 不太稳定，谨慎使用
- ❌ 一致性 < 40%，平均测试收益为负 → 这个策略在这只股票上**不靠谱**

## 11. 常用操作流程

### 日常看盘流程

```
1. 打开程序
2. 在自选股页面一眼扫：
   - 有没有"强烈买入"或"强烈卖出"的信号？
   - 涨跌幅是否异常？
3. 点击感兴趣的股票进入详情
4. 看右侧"多空评分"和"关键价位"
5. 看K线图和MACD判断趋势
```

### 想买入一只新股票

```
1. 搜索股票，进入详情页
2. 看"多空评分" — 多头 ≥ 4 说明当前偏多
3. 看"关键价位" — 记下"建议买入价"
4. 做回测 — 看该策略在这只股票的历史表现
5. 做Walk-Forward — 确认策略是否靠谱
6. 如果回测赚钱、WF一致性>60%，可以考虑按"建议买入价"附近买入
7. 记下"止损价位"，亏到这个价就卖
```

### 手里有股票，想判断要不要卖

```
1. 进入该股票的详情页
2. 如果信号出现"卖出"或"强烈卖出" → 注意风险
3. 看RSI — 如果 > 70，说明偏热
4. 看MACD — 如果柱子由红变绿，趋势可能转弱
5. 看"关键价位" → "建议卖出价"可以作为参考
```

## 12. 参数说明（进阶）

> 这部分是进阶内容，日常使用不需要改这些。默认参数就够用了。

### 回测参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 初始资金 | 100万 | 模拟你有多少钱开始炒 |
| 佣金费率 | 0.00025 | 每次买卖的手续费，万分之2.5 |
| 印花税率 | 0.0005 | 卖出时国家收的税，万分之5 |
| 滑点(基点) | 0 | 模拟买入价偏高/卖出价偏低（0=不模拟） |
| 最低佣金 | 5元 | 每笔交易手续费最少5块钱 |
| 回撤熔断 | 0% | 亏到多少就不再买了（0=不限制，建议设20%） |
| 样本外 | 0% | 留多少数据做测试（0=不做，建议设30%） |

### 止盈止损

| 参数 | 说明 | 建议 |
|------|------|------|
| 止损(%) | 亏多少就卖 | 5%–10% |
| 止盈(%) | 赚多少就卖 | 10%–20% |
| 移动止损(%) | 从最高点回落多少就卖 | 5%–8% |
| ATR止损(倍) | 用波动率计算止损 | 2.0（进阶用户） |

### 仓位模式

| 模式 | 说明 | 适合 |
|------|------|------|
| 全仓 | 每次全部买入 | 简单但风险大 |
| 固定比例 | 每次只买一部分 | 稳健，建议50-80% |
| 凯利公式 | 根据历史胜率自动计算 | 最科学，交易次数多后效果好 |

## 13. 常见问题

### Q: 程序打不开怎么办？
**A:** 可能被杀毒软件拦截了。右键 → "以管理员身份运行"，或者把程序加入杀毒软件白名单。

### Q: 数据加载很慢？
**A:** 第一次获取某只股票的数据会慢一些（从网上下载），之后会缓存，第二次就快了。如果一直很慢，检查网络连接。

### Q: 信号说买入，我买了就跌了！
**A:** 信号是基于历史数据的概率判断，不是100%准确。**永远要设止损**，控制风险比追求收益更重要！

### Q: 回测说胜率80%，为什么实际没那么好？
**A:** 回测是在历史数据上"事后诸葛亮"，实际操作有滑点、手续费、心态影响。建议：
- 看Walk-Forward验证的结果更靠谱
- 一致性评分 > 60% 的策略才值得用
- 回测收益打个5-7折来预期

### Q: 应该用哪个策略？
**A:** 
- 不知道选什么 → 用默认的 **"MACD金叉+RSI超买"**
- 想稳一点 → 用 **"稳健保守"**
- 想简单一点 → 用 **"纯MACD交叉"** 或 **"纯EMA交叉"**
- 最好用 **策略对比** 功能，找出在你关注的股票上表现最好的策略

### Q: 深色模式怎么切换？
**A:** 点右上角的 **☀**（太阳）或 **🌙**（月亮）图标。

### Q: 手机上显示不全？
**A:** 手机横屏看图表效果更好。信息栏在手机上会自动排到图表下方。

### Q: 实时价格为什么不动？
**A:** 只有在A股交易时间（周一至周五 9:30-15:00）才有实时数据。收盘后显示的是收盘价。
